# cnode-react

> cnode社区论坛的react版本

> 使用 [umijs](https://umijs.org/zh/) + [dva](https://dvajs.com/) + [ant-mobile](https://mobile.ant.design/)

> [测试版](https://cnode.wbget.com/)

## Build Setup

``` bash
# install dependencies
npm install

# get start
npm start
```
